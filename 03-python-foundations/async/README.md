# Async examples (`asyncio`)

These scripts mirror ideas from **`../exercises/09-async-io.ipynb`** and **`../exercises/18-asyncio-queue-pipelines.ipynb`**. Run them from **`03-python-foundations`** so imports stay simple:

```bash
cd 03-python-foundations
python async/async_gather_demo.py
python async/async_queue_demo.py
```

No third-party packages — stdlib **`asyncio`** only.
